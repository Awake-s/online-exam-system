# 在线考试系统测试文档

## 一、测试环境

| 项目 | 说明 |
|------|------|
| 后端 | Spring Boot 2.7.18 + MyBatis-Plus + Spring Security + JWT |
| 前端 | Art Design Pro + Vue3 + TypeScript + Vite + Element Plus + Tailwind CSS + Pinia + Axios + ECharts |
| 数据库 | MySQL 8.0 |
| 后端端口 | 8080 |
| 前端端口 | 3000 |
| 浏览器 | Chrome / Edge 最新版 |

## 三、启动步骤

### 3.1 启动后端

```bash
cd exam-system
mvn spring-boot:run
```

### 3.2 启动前端

```bash
cd art-design-pro-ui
npm run dev
```

### 3.3 访问地址

浏览器打开 `http://localhost:3000`
