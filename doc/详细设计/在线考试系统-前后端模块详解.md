# 在线考试系统 — 前后端模块详解

> **技术栈**：后端 Spring Boot 2.7.18 + MyBatis-Plus 3.5.5 + Spring Security + WebSocket（STOMP + SockJS） | 前端 Vue3 + TypeScript + Element Plus + Tailwind CSS  
> **Java 版本**：11 | **构建工具**：Maven | **数据库**：MySQL 8.0 | **缓存**：Redis | **Excel处理**：EasyExcel 3.3.3 | **工具库**：Hutool 5.8.25

---

## 一、后端项目结构（exam-system）

### 1.1 总体目录

```
exam-system/src/main/java/com/exam/
├── ExamApplication.java          # Spring Boot 启动入口（端口8080）
├── common/                       # 公共基础设施层
├── config/                       # WebSocket 配置层
├── security/                     # Spring Security 安全认证层
├── entity/                       # 数据库实体类（19张表）
├── mapper/                       # MyBatis-Plus 数据访问接口（19个）
├── dto/request/                  # 前端请求参数封装（15个）
├── controller/                   # REST API 控制器（17个REST + 1个WebSocket + 1个WS消息 = 19个）
├── service/                      # 业务接口（20个）+ 实现类（17个）
└── task/                         # 定时任务（2个）
```

### 1.2 common — 公共基础设施

| 文件 | 模块 | 作用 |
|------|------|------|
| `config/DataInitializer.java` | 系统初始化 | 首次启动自动创建管理员账号 |
| `config/MyBatisPlusConfig.java` | 数据访问 | 分页插件配置，所有列表分页查询依赖此 |
| `config/WebMvcConfig.java` | 文件服务 | 将 uploads/ 目录映射为URL，支持头像访问 |
| `constants/RoleConstants.java` | 权限 | 定义 ADMIN/TEACHER/STUDENT 三种角色常量 |
| `exception/BusinessException.java` | 全局 | 自定义业务异常（如"考试已结束"） |
| `exception/GlobalExceptionHandler.java` | 全局 | 统一异常拦截，返回 `{code, message, data}` JSON |
| `exception/NotFoundException.java` | 全局 | 资源未找到异常 |
| `result/Result.java` | 全局 | 统一响应格式 `{code, message, data}`，code=200 表示成功 |
| `result/PageResult.java` | 全局 | 分页响应格式 `{records, total, page, size, pages}` |
| `utils/JwtUtils.java` | 认证 | JWT Token 生成/解析/验证工具 |

### 1.3 config — WebSocket 配置

| 文件 | 作用 |
|------|------|
| `WebSocketConfig.java` | 配置 STOMP 代理；两个端点 `/ws`（聊天）和 `/ws/notification`（通知）；消息代理 `/topic`、`/queue`；应用前缀 `/app`；用户前缀 `/user` |
| `WebSocketAuthInterceptor.java` | 握手时从URL提取JWT验证用户身份 |
| `StompChannelInterceptor.java` | 消息通道权限拦截 |
| `CustomHandshakeHandler.java` | 将用户ID设为 Principal，实现点对点推送 |
| `WebSocketEventListener.java` | 监听连接/断开事件，跟踪在线状态 |

### 1.4 security — 安全认证

| 文件 | 作用 |
|------|------|
| `SecurityConfig.java` | 安全过滤链：放行登录接口，其余需Token；CORS配置 |
| `JwtAuthenticationFilter.java` | 每个请求提取Token → 验证 → 设置SecurityContext |
| `UserDetailsServiceImpl.java` | 从数据库加载用户信息及角色 |
| `LoginUser.java` | 封装登录用户主体（ID/用户名/角色） |
| `SecurityUtils.java` | 静态方法 `getCurrentUserId()` 获取当前用户 |
| `TokenBlacklistService.java` | 登出时Token加入黑名单 |
| `LoginRateLimiter.java` | 基于用户名限制登录频率（5次失败锁定15分钟），防暴力破解 |

## 四、数据流转示意

以 **学生参加考试** 为例，展示完整的前后端调用链路：

```
学生点击"开始考试"
  └→ 前端 exam-page/index.vue
      └→ api/exam/studentExam.ts → startExam(examId)
          └→ POST /api/student/exam/start/{examId}
              └→ StudentExamController.start()
                  └→ StudentExamServiceImpl.startExam()
                      ├→ ExamExamMapper: 查询考试信息
                      ├→ ExamPaperQuestionMapper: 加载试卷题目
                      ├→ ExamRecordMapper: 创建考试记录
                      └→ 返回试题数据给前端

学生答题过程（每30秒自动保存）
  └→ api/exam/studentExam.ts → autoSave(data)
      └→ POST /api/student/exam/auto-save
          └→ StudentExamServiceImpl.autoSaveAnswers()
              └→ ExamAnswerMapper: upsert 每题答案

学生点击"交卷"
  └→ api/exam/studentExam.ts → submitExam(data)
      └→ POST /api/student/exam/submit
          └→ StudentExamServiceImpl.submitExam()
              ├→ ExamAnswerMapper: 保存最终答案
              ├→ 自动批改客观题（单选/多选/判断）
              ├→ ExamRecordMapper: 更新状态为"已交卷"
              └→ NotificationService: 发送交卷通知给教师
```

以 **教师阅卷发布成绩** 为例：

```
教师进入阅卷详情
  └→ 前端 marking-detail/index.vue
      └→ api/exam/marking.ts → getMarkingDetail(recordId)
          └→ GET /api/marking/detail/{recordId}
              └→ MarkingServiceImpl.getMarkingDetail()
                  └→ 加载学生每题答案 + 正确答案 + 题目详情

教师逐题打分并提交
  └→ api/exam/marking.ts → submitScore(data)
      └→ POST /api/marking/score
          └→ MarkingServiceImpl.markScores()
              ├→ ExamAnswerMapper: 更新每题得分
              └→ ExamRecordMapper: 更新总分+状态为"已批改"

教师发布成绩
  └→ api/exam/marking.ts → publishScores(examId)
      └→ POST /api/marking/publish/{examId}
          └→ MarkingServiceImpl.publishScores()
              ├→ ExamRecordMapper: 更新状态为"已发布"
              ├→ NotificationService: 给每个学生发成绩通知
              └→ NotificationPushService: WebSocket实时推送
                  └→ 学生浏览器通知铃铛实时显示新通知
```
