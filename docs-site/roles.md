# 三端展示

> 一个系统，三种角色，权限严格隔离。基于角色码驱动的动态路由，同一 SPA 内按角色渲染差异化界面。

## 角色概览

| 角色 | 角色码 | 职责定位 |
| :--- | :--- | :--- |
| **管理员** | `ADMIN` | 系统运维：用户 / 专业 / 班级 / 科目的全量管理 |
| **教师** | `TEACHER` | 教学业务：题库 · 组卷 · 发布考试 · 阅卷 · 成绩分析 |
| **学生** | `STUDENT` | 参考主体：在线考试 · 查成绩 · 错题本 · 师生交流 |

## 管理员端

管理员负责系统的用户与教务基础数据维护，是平台的「后台管理者」。

**核心模块**：

- **首页仪表盘**：用户数 / 考试数 / 试卷数等关键指标概览
- **用户管理**：管理员 / 教师 / 学生账号 CRUD，支持 Excel 批量导入
- **专业管理**：院系专业维护，关联班级与科目
- **班级管理**：班级 CRUD，关联专业，分配学生
- **科目管理**：考试科目维护，建立科目-专业-教师的任课关系

::: details 权限实现
后端 `@PreAuthorize` 类级 / 方法级控制（共 44 处注解覆盖 14 个 Controller，其中 ADMIN 专属 11 处）；前端角色码 `ADMIN` 映射路由角色 `R_ADMIN`，动态注册管理员专属路由，侧边栏按角色过滤菜单。
:::

## 教师端

教师是教学业务的核心执行者，覆盖出题到阅卷的完整教学闭环。

**核心模块**：

- **题库管理**：5 种题型的增删改查，富文本编辑（支持公式 / 图片），Excel 批量导入导出
- **试卷管理**：手动组卷 + 模板自动组卷
- **试卷模板**：配置抽题规则（题型 / 难度配比），一键生成平行卷
- **考试管理**：发布考试，关联试卷与班级，设定时间 / 时长 / 状态
- **阅卷**：客观题自动批改，主观题在线评分 + 批注
- **成绩管理**：成绩查询 / 导出，支持成绩复核
- **成绩分析**：班级成绩统计、正确率分析、分数分布、趋势图

::: details 组卷能力
支持两种组卷方式：
1. **手动组卷**：从题库逐题挑选，精确控制每道题
2. **模板自动组卷**：设定 `exam_paper_template` + `exam_template_rule`，按规则随机抽题，可批量生成多套等难度平行试卷
:::

## 学生端

学生是考试的参与者，提供沉浸式考试体验与学习辅助工具。

**核心模块**：

- **首页**：待考 / 进行中 / 已完成的考试列表，通知中心
- **我的考试**：按状态（待考 / 进行中 / 已结束）查看可参加的考试
- **在线答题**：全屏答题界面，倒计时、自动保存、切屏检测、到时自动交卷
- **我的成绩**：历次考试成绩，支持成绩详情（逐题得分）
- **错题本**：自动归集错题，按题型筛选，支持重做
- **即时通讯**：与任课教师一对一聊天（基于 STOMP WebSocket）

::: details 防作弊机制
- 进入考试强制全屏，退出触发警告并记录
- 切屏次数检测，超过阈值标记违纪或自动交卷
- 剩余时间由服务端 `exam_exam.end_time` + 定时任务 `ExamEndTask` 双重校验，到时强制收卷
- 答题过程定时调用保存接口持久化 `exam_answer`，断网 / 刷新可基于已保存数据续答
:::

---

## 共享模块

所有角色通用：

- **个人中心**：修改个人信息 / 头像 / 密码
- **通知中心**：12 类通知（考试发布、交卷提醒、成绩公布等），WebSocket 实时推送，已读 / 未读管理
- **即时聊天**：师生一对一交流，在线状态感知

---

::: tip 技术实现
三端共享同一 Vue3 SPA（`art-design-pro-ui`），通过**角色码驱动的动态路由**实现差异化：登录后 `fetchUserInfo()` 拉取用户 `roleCode`，映射为路由角色，`RouteRegistry` 动态注册该角色可访问的路由，`MenuProcessor` 过滤侧边栏菜单。未登录访问任意业务路径统一重定向到登录页。
:::
