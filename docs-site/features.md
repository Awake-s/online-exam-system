## 亮点能力详解

### 智能组卷（模板自动组卷）

教师设定抽题规则（题型配比、难度分布、知识点覆盖范围、总分与分值），系统按规则从题库随机抽取，**一键生成多套平行试卷**，避免学生间抄袭。

```
模板规则示例：
├─ 单选题：10 道 × 2 分（难度：易 4 / 中 4 / 难 2）
├─ 多选题：5 道 × 4 分（难度：中 3 / 难 2）
├─ 判断题：10 道 × 1 分（难度：易）
├─ 填空题：5 道 × 2 分（难度：中）
└─ 简答题：2 道 × 10 分（难度：难）
合计：100 分
```

### 在线考试防作弊

- **全屏模式**：进入考试强制全屏，退出全屏触发警告
- **切屏检测**：记录切屏次数，超过阈值自动交卷或标记违纪
- **倒计时控制**：服务端校验剩余时间，到时强制交卷（防客户端篡改）
- **答案自动保存**：答题过程中定时持久化，**断网 / 刷新可续答**

### 自动阅卷与成绩汇总

- 客观题（单选 / 多选 / 判断 / 填空）：提交即自动评分
- 主观题（简答）：教师在线评分，支持批注
- 成绩自动汇总：客观分 + 主观分，支持成绩复核与重新发布

### 多维成绩分析

基于 ECharts 6 提供的可视化：

- **班级成绩概览**：最高分 / 最低分 / 平均分 / 及格率 / 标准差
- **分数段分布**：直方图展示各分数段人数
- **题目正确率**：逐题正确率，定位教学薄弱点
- **成绩趋势**：多次考试成绩对比折线图
